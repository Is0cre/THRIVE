# THRIVES — Monetisation Strategy

Core principle: The app is free. The builder gets paid. These are not in conflict.

---

## What Will Never Be Monetised
- Core features — free forever
- Crisis and grounding tools — never paywalled
- User data — never sold, never collected
- Ads — never

---

## Revenue Streams (Non-Conventional)

### 1. NLnet Foundation Grant — HIGHEST PRIORITY
- nlnet.nl — funds privacy-preserving open source tools
- THRIVES fits their criteria precisely
- Grants typically €5,000–€50,000
- No equity, no strings, no compromise of values
- Action: Check open calls, prepare application

### 2. NGO / Charity White-Label Consulting
- Refugee support organisations
- Domestic violence charities
- Veteran support groups
- Addiction recovery organisations
- They have grants, they need tools like this
- You don't sell the app — you sell deployment, customisation, support
- Typical engagement: €5,000–€20,000 per organisation

### 3. Dual Licensing
- Individual use: GPL — free forever
- Institutional use (clinics, corporates, governments wanting proprietary integration): Commercial licence
- They pay to use it without open sourcing their stack
- Individuals always free

### 4. Psychology Professional Tier
- Small annual fee for therapists/practitioners
- Branded version to recommend to clients
- Session export formatted for clinical notes
- Bulk deployment tools for their practice
- Never affects individual user access

### 5. Open Collective / GitHub Sponsors
- Transparent donation platform
- Full financial visibility — builds trust
- Named supporter credits in app (optional, privacy-respecting)
- Framing: "This app is free because X people believe in it"

### 6. Speaking and Writing
- Conferences: trauma tech, mental health tech, privacy, open source health
- Unique position: lived experience + technical builder + privacy architecture
- Paid speaking once the app has traction

### 7. Hardware Partnerships (Future)
- Wearable companies (Polar, Garmin, smaller biofeedback makers)
- Featured integration partnerships
- You control the terms — no compromising privacy
